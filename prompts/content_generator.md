\# AI内容生成器使用指南

\## 简介

本文件是AI内容生成器的模板库，用于快速产出各类短视频/图文内容。



\## 使用方法

\### 基础调用

请基于\[主题],生成目标内容平台(TikTok/小红书/抖音等)内容包:

1\. 短视频脚本(XX秒)

2\. 封面文案(X个版本)

3\. 标题(X个A/B测试版本)

4\. Hashtag策略



\### 模板变量

| 变量 | 说明 | 示例 |

|------|------|------|

| \[主题] | 内容核心话题 | "AI工具转行经历" |

| \[平台] | 目标平台 | TikTok/小红书/抖音 |

| \[目标受众] | 目标用户画像 | 18-30岁转行者 |

| \[账号定位] | 账号人设 | "AI转行真实案例分享" |



\### 输出格式

生成内容默认输出JSON格式，便于程序化处理:

```json

{

&#x20; "script": "...",

&#x20; "covers": \["...", "..."],

&#x20; "titles": \["...", "..."],

&#x20; "hashtags": {

&#x20;   "hot": \["...", "..."],

&#x20;   "niche": \["...", "..."],

&#x20;   "brand": "..."

&#x20; }

}
