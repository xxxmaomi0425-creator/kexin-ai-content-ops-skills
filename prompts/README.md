\# kexin-ai-content-ops-skills

45天AI内容运营转行自动化工具箱



\## 仓库简介

这是一个专注于AI内容运营自动化的个人工具仓库，帮助我通过Claude Code + n8n/Coze + Git Skill快速产出爆款内容，实现45天转行AI内容运营专员的目标。



\## 目录结构

kexin-ai-content-ops-skills/

├── prompts/                  # Prompt模板库

│   ├── Prompt模板V1.md

│   ├── content\_generator.md

│   └── trend\_analyzer.md

├── workflows/                # n8n/Coze自动化工作流

├── scripts/                  # Python/JS脚本

├── 爆款拆解/                 # 爆款案例拆解记录

├── 作品集/                   # 最终产出的内容作品

└── README.md



\## 快速开始

1\. 使用Prompt模板  

&#x20;  打开 `prompts/Prompt模板V1.md`，填写变量后直接复制到Claude/Grok使用。



2\. 爆款拆解流程  

&#x20;  使用 `trend\_analyzer.md` 进行结构化拆解 → 用Prompt模板生成同款内容。



3\. 自动化工作流  

&#x20;  后续将在 `workflows/` 目录存放 n8n/Coze JSON 配置。



\## 核心工具

\- Claude Code + Hermes Agent

\- n8n / Coze（无代码工作流）

\- Git（版本管理 + Skill调用）

\- Midjourney / Kling（视觉素材）
